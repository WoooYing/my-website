HTML 基本語法

HTML（超文本標記語言）用於建立網頁的結構。以下是一個基本範例：

<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8">
    <title>我的網頁</title>
</head>
<body>
    <h1>歡迎來到我的網站</h1>
</body>
</html>

CSS 基本語法

CSS（層疊樣式表）用於設計網頁的外觀。以下是一個基本範例：

body {
    font-family: Arial, sans-serif;
    background-color: #ffffff;
    color: #333333;
    margin: 0;
    padding: 0;
}

h1 {
    font-size: 2em;
    color: #000000;
}
